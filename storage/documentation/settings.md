# Settings & Preferences

Configure your account, notifications, and preferences. Learn how to customize Nestogy to match your workflow and business needs.

## Accessing Settings

Navigate to settings:

1. Click your profile picture/name in the top right
2. Select **Settings** from the dropdown
3. Browse settings categories in the left sidebar
4. Make changes as needed
5. Click **Save** to apply changes

## Best Practices

Tips for optimal configuration:

### Security

- Enable 2FA for all users
- Set strong password policies
- Restrict IP access when possible
- Review audit logs regularly
- Keep integrations updated

### Organization

- Establish naming conventions
- Document custom configurations
- Train team on settings
- Review settings quarterly
- Keep backups current

### Performance

- Optimize email rules
- Limit automation complexity
- Archive old data
- Monitor storage usage
- Regular cleanup
