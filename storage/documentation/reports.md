# Reports & Analytics

Generate reports, analyze data, and create custom dashboards. Learn how to access built-in reports, create custom reports, schedule automated delivery, and gain insights into your MSP business.

## Understanding Reports

Nestogy provides comprehensive reporting across all modules:

- Financial performance
- Service delivery metrics
- Team productivity
- Client analytics
- Contract compliance
- Asset inventory

---

## Accessing Reports

Navigate to reports:

1. Click **Reports** in the sidebar
2. Browse report categories
3. Select a report
4. Configure parameters
5. Run or schedule report

## Custom Reports

Build your own reports:

### Report Builder

Create custom reports:

1. Navigate to Reports â†’ Custom Reports
2. Click **Create Report**
3. Select data source (tickets, time, invoices, etc.)
4. Choose fields to include
5. Add filters and conditions
6. Configure grouping and sorting
7. Select visualization (table, chart, etc.)
8. Save and name report

### Filters and Parameters

Refine report data:

**Date Filters:**
- Date range
- Relative dates (last 7 days, this month)
- Comparison periods
- Fiscal year alignment

**Entity Filters:**
- Specific clients
- Team members
- Ticket categories
- Project types
- Service lines

**Status Filters:**
- Open/closed
- Active/inactive
- Billable/non-billable
- Approved/pending

### Calculated Fields

Create custom metrics:

- Average values
- Percentages and ratios
- Cumulative totals
- Growth rates
- Custom formulas

## Scheduled Reports

Automate report delivery:

### Setting Up Schedules

1. Open any report
2. Click **Schedule**
3. Configure schedule:
   - Frequency (daily, weekly, monthly)
   - Day and time
   - Recipients
   - Format (PDF, Excel, CSV)
4. Save schedule

### Report Distribution

Automatic delivery options:

- Email to recipients
- Save to shared folder
- Upload to cloud storage
- Post to webhook
- API integration
