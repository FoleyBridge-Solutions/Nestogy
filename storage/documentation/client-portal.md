# Client Portal

Set up and manage the client portal for your customers. Learn how to enable portal access, customize the experience, and empower clients with self-service capabilities.

## Understanding the Client Portal

The client portal gives your customers 24/7 access to:

- Submit and track support tickets
- View and pay invoices
- Monitor service agreements
- Access knowledge base articles
- View asset inventory
- Download reports
- Communicate with your team

---

## Enabling the Portal

Activate the client portal:

1. Navigate to Settings → Client Portal
2. Toggle **Enable Client Portal**
3. Configure portal URL:
   - Subdomain: `portal.yourcompany.com`
   - Custom domain (optional)
4. Click **Save Settings**

## User Management

Control who can access the portal:

### Inviting Users

Send portal invitations:

1. Navigate to client's detail page
2. Click **Portal Access** tab
3. Select contacts to invite
4. Click **Send Invitation**
5. Contacts receive email with setup link

### User Roles

Different permission levels:

**Administrator:**
- Full portal access
- Manage other portal users
- View all client data
- Submit tickets for anyone
- Access financial information

**Standard User:**
- Submit own tickets
- View assigned tickets
- See own invoices
- Limited asset access
- Basic reporting

**Read-Only:**
- View tickets only
- View invoices only
- No submission capability
- Reports only

### Multi-Factor Authentication

Enhance portal security:

- Enable 2FA requirement
- Email verification codes
- Authenticator app support
- SMS verification
- Backup codes

## Knowledge Base

Self-service content library:

### Creating Articles

Build your knowledge base:

1. Navigate to Portal → Knowledge Base
2. Click **Create Article**
3. Write article content:
   - Title
   - Category
   - Content (rich text)
   - Attachments
   - Keywords/tags
4. Set visibility (all clients or specific)
